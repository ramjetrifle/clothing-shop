# Clothing Shop E-Commerce App

A fullstack e-commerce web application built with Spring Boot and Thymeleaf.

## Technologies Used

### Backend
- **Spring Boot** 
- **Spring Security**
- **Spring Data JPA**
- **H2 Database**
- **PostgreSQL**
- **JWT**
- **Thymeleaf**
- **Bootstrap 5** 
- **JavaScript**

## Installation

```
git clone https://github.com/ramjetrifle/clothing-shop.git
```
